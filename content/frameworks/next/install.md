## Clone Existing App

```bash
git clone [git-repo-url]
cd [git-repo]
npm install
npm install next@latest react@latest react-dom@latest
```

## Run Dev Svr

```bash
npm run dev  # URL: http://localhost:3000
```

## First Impressions
