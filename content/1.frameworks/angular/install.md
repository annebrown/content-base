---

## Install Angular CLI

```bash
npm install -g @angular/cli@llatest
```

## Create New Web App

```bash
sudo npm install -g @angular/cli@latest
ng new mangular
Autocompletion?n
Share usage data? n
Stylesheet fm: css for now
SSR & SSG? y

...

Successfully initialized git.
```

## Run App

```bash
cd mangular
ng serve --open
```
